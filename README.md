# Quicksilver Reading Companion

A page-by-page reading companion for Neal Stephenson's *Quicksilver* (The Baroque Cycle, Vol. 1). Historical context, scientific background, and topic pages to enrich your reading — no plot spoilers.

**Live at [read.quicksilver.wiki](https://read.quicksilver.wiki)**

## What's in it

- **68 annotated chapters** (1,500+ notes across all three books) covering real people, events, science, and places
- **110+ topic pages** on characters, historical figures, institutions, and concepts
- **"Before You Read" primers** for each book with historical context
- Annotations from Stephenson himself and the original community wiki

## Development

Built with [Astro](https://astro.build/).

```bash
npm install
npx astro dev
```

## Build

